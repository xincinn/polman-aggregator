# polman-aggregator
Aggregator Portal Digital Politeknik Manufaktur Bandung (POLMAN). Platform terpusat untuk mengakses layanan akademik, LMS, kepegawaian, dan helpdesk kampus dalam satu pintu.

# Pusat Layanan Digital - Polman Bandung 🎓

Sebuah *landing page aggregator* yang dirancang untuk mengumpulkan, mengorganisasi, dan menyajikan seluruh tautan layanan digital di lingkungan Politeknik Manufaktur Bandung (POLMAN) ke dalam satu platform terpusat. 

Proyek ini bertujuan untuk memudahkan akses civitas akademika menuju berbagai sistem kampus melalui antarmuka yang bersih, modern, dan minimalis.

## Fitur Utama
- **Akses Terpusat 6 Aplikasi Kampus:** Tautan langsung menuju sistem Akademik, Kepegawaian, Keuangan, Logistik, Panduan (SOP), dan LMS Belajar.
- **Pusat Informasi:** Rangkuman berita terbaru dan kalender agenda kegiatan akademik kampus.
- **Helpdesk Terintegrasi:** Tombol pintasan untuk membuat tiket bantuan IT yang terhubung langsung ke WhatsApp.
- **Desain Responsif:** Tampilan yang sudah dioptimalkan untuk pengguna *desktop*, *tablet*, maupun *mobile* (smartphone).

## Teknologi yang Digunakan
- **HTML5:** Struktur dasar halaman *website*.
- **Tailwind CSS:** *Framework* CSS untuk tata letak (*layouting*), warna (*color palette* resmi kampus), dan efek responsif tanpa memerlukan file CSS eksternal.
- **FontAwesome:** Sistem ikon minimalis.
- **Google Fonts (Poppins):** Tipografi standar yang modern dan mudah dibaca.
